ΣΥΝΕΙΣΦΟΡΑ

Το EasySubs είναι ένα έργο ανοιχτού κώδικα που ευδοκιμεί χάρη στις συνεισφορές της κοινότητας.

Ελάτε μαζί μας στη βελτίωση αυτού του εργαλείου αναπτύσσοντας νέα χαρακτηριστικά ή βελτιώνοντας την υποστήριξη για τη μητρική σας γλώσσα και βοηθήστε να κάνουμε την εκμάθηση γλωσσών μέσω βίντεο ακόμα καλύτερη.

Η εμπειρογνωμοσύνη σας μπορεί να προωθήσει το EasySubs, καθιστώντας το ένα πιο ισχυρό και ευέλικτο εργαλείο για μαθητές γλωσσών παγκοσμίως.
